# RAG-LangChain-Project
Retrieval Augmented Generation (RAG) project using LangChain and GeminiAI in Google Colab.

RAG with LangChain

This project demonstrates a Retrieval-Augmented Generation (RAG) pipeline using LangChain.

## Features
- Document Loading
- Text Splitting
- Vector Embeddings
- Vector Database (Chroma)
- Retrieval QA
- LLM Response Generation

## Technologies Used
- Python
- LangChain
- ChromaDB
- GeminiAI / LLM
- Google Colab

## Workflow
1. Load documents
2. Split text into chunks
3. Generate embeddings
4. Store in vector database
5. Retrieve relevant documents
6. Generate answer using LLM
